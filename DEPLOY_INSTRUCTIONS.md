# تعليمات نشر نظام الحضور على Vercel مع التخزين السحابي

## الخطوة 1: رفع المشروع على GitHub

```bash
cd presence
git add .
git commit -m "Add cloud storage with Vercel KV"
git push
```

## الخطوة 2: إنشاء مشروع على Vercel

1. اذهب إلى [vercel.com](https://vercel.com)
2. سجل دخول بحساب GitHub
3. اضغط **"Add New Project"**
4. اختر مشروعك من القائمة
5. اضغط **Deploy** (انتظر حتى ينتهي)

## الخطوة 3: إضافة Vercel KV (التخزين السحابي)

هذي الخطوة **مهمة جداً** - بدونها لن يعمل التخزين!

1. من لوحة تحكم مشروعك في Vercel
2. اذهب إلى تبويب **"Storage"**
3. اضغط **"Create Database"**
4. اختر **"KV"** (مجاني)
5. اختر اسم مثل `presence-db`
6. اختر المنطقة الأقرب لك
7. اضغط **"Create"**

## الخطوة 4: ربط KV بالمشروع

1. بعد إنشاء الـ KV، ستظهر لك صفحة الإعدادات
2. اضغط **"Connect to Project"**
3. اختر مشروعك
4. اضغط **"Connect"**

## الخطوة 5: إعادة النشر

بعد ربط KV، تحتاج إعادة نشر:

1. اذهب إلى تبويب **"Deployments"**
2. اضغط على آخر deployment
3. اضغط **"Redeploy"**
